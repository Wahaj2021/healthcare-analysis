# Healthcare Data Analysis (Python & Power BI)

##  Business Problem

Healthcare organisations rely on data to improve patient outcomes, optimise resource allocation, and reduce operational inefficiencies.

This project analyses healthcare-related data to uncover trends, patterns, and insights that can support better decision-making.

---

##  Objectives

- Analyse patient and healthcare metrics
- Identify trends and patterns in the data
- Highlight key factors impacting outcomes
- Build an interactive dashboard for monitoring insights
- Provide data-driven recommendations

##  Dataset Overview

The dataset includes variables related to:
- Patient demographics
- Medical or service-related attributes
- Outcome indicators

---

##  Data Cleaning & Preparation

- Handled missing and inconsistent values
- Removed duplicate records
- Standardised column names and formats
- Converted date fields into usable time features

---

##  Key Analyses

- Distribution of key healthcare metrics
- Trend analysis over time
- Demographic-based comparisons
- Identification of high-risk or high-impact groups

---

##  Key Insights

- Certain demographic groups show higher risk or impact levels.
- Trends over time indicate periods of increased demand.
- Key variables strongly influence healthcare outcomes.
- Data highlights opportunities for preventive interventions.
